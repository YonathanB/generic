  • Devices that communicate using HTTP, include Lua code, use fractional number states, or require logging-in may not operate correctly.
  • Not all UI elements are fully supported: Partial support for the slider, frames/overlays/pages, lists, and non-interactive foreground elements; Background does not support text and cannot be changed during runtime; Foreground element texts and fonts cannot be modified.   • The following system device states are supported: "TIME_STATE", "DATE_STATE", "HOUR_STATE", "MINUTE_STATE", "SECOND_STATE", "DAY_OF_WEEK_STATE", "DAY_OF_MONTH_STATE", "MONTH_STATE", "YEAR_STATE".
